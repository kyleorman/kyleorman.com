The capstone project goal was to build an automatic reordering system that could predict what items a store needs before they are depleted, reducing delays and avoiding costly shipping fees.

Public project documentation identifies Kyle’s responsibilities as machine learning research and implementation. Weekly updates and task records on the capstone site describe work on the TensorFlow and Keras pipeline, testing, refinement, and documentation for the machine learning portion of the project.
